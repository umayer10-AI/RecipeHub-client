# 🍳 RecipeHub

RecipeHub is a modern recipe-sharing platform where users can discover, create, save, and manage delicious recipes. The platform provides an intuitive interface for exploring recipes by category, cuisine type, and difficulty level while allowing authenticated users to contribute their own recipes.

## 🚀 Features

### 👤 Authentication

* Secure user registration and login
* JWT-based authentication
* Protected routes for authorized users

### 🍽️ Recipe Management

* Add new recipes
* Update existing recipes
* Delete recipes
* View recipe details
* Browse all available recipes

### ❤️ Save Recipes

* Save favorite recipes
* View saved recipes
* Remove recipes from saved list

### 🔍 Search & Filter

* Search recipes by name
* Filter by category
* Filter by cuisine type
* Filter by difficulty level

### 📊 Dashboard

* User profile management
* Personal recipe management
* Saved recipes overview

### 🎨 Modern UI

* Responsive design
* Dark/Light theme support
* Interactive user experience
* Toast notifications

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/umayer10-AI/RecipeHub-client.git
```

### Navigate to Project

```bash
cd recipehub-client
```

### Install Dependencies

```bash
npm install
```

### Environment Variables

Create a `.env.local` file:

```env
NEXT_PUBLIC_API_URL=your_backend_api_url
NEXT_PUBLIC_BASE_URL=your_frontend_url
```

### Run Development Server

```bash
npm run dev
```

Open:

```bash
https://recipe-hub-client-chi.vercel.app
```
